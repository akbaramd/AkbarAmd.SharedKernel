# Documentation Setup

## Installation

Docsify is already installed. To serve the documentation locally:

```bash
cd docs
npm run serve
```

Then open http://localhost:3000 in your browser.

## Documentation Structure

- `README.md` - Home page and overview
- `getting-started.md` - Quick start guide
- `basic-usage.md` - Basic usage examples
- `fluent-api.md` - Fluent API documentation
- `database-patterns.md` - Database and EF Core patterns
- `enterprise-patterns.md` - Enterprise patterns and best practices
- `advanced-examples.md` - Advanced usage examples
- `api-reference.md` - Complete API reference

## Adding New Documentation

1. Create a new `.md` file in the `docs` folder
2. Add it to `_sidebar.md` for navigation
3. Update `README.md` if needed

## Publishing

The documentation can be published to GitHub Pages or any static hosting service.
